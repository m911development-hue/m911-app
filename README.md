# Metaverse Technologies

A comprehensive educational website exploring Metaverse Technologies including Virtual Reality (VR), Augmented Reality (AR), Mixed Reality (MR), and the Future of the Metaverse.

## Tech Stack

- **React.js** - Frontend library
- **Vite** - Build tool
- **React Router DOM** - Client-side routing
- **CSS** - Custom styling with futuristic design

## Getting Started

```bash
npm install
npm run dev
```

## Features

- 🌐 Comprehensive Metaverse education content
- 🎮 Virtual Reality deep dive
- 📱 Augmented Reality exploration
- 🥽 Mixed Reality analysis
- 🔮 Future of Metaverse predictions
- 🎬 Video learning system
- ✨ Futuristic glassmorphism UI
- 📱 Fully responsive design
- 🎨 Animated hero sections
- 🔄 Smooth page transitions

## Pages

1. **Home** - Introduction to Metaverse
2. **Virtual Reality** - VR technologies and applications
3. **Augmented Reality** - AR systems and use cases
4. **Mixed Reality** - MR devices and enterprise solutions
5. **Future of Metaverse** - AI, Blockchain, NFTs, and 2035 predictions
